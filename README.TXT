====================================
🐒 JUNGLE ESCAPE - Game Instructions
====================================

HOW TO PLAY:
------------
Monkey runs automatically to the right!
You only need 2 buttons: JUMP and SLIDE

Press JUMP (⬆️) to jump over:
   • Snakes
   • Crocodiles
   • Logs
   • Lava pits

Press SLIDE (⬇️) to slide under:
   • Hanging branches
   • Stalactites

COLLECT BANANAS 🍌 for points!
Every 10 bananas = 1 extra life

SECRET FEATURES:
----------------
• Rope Swings: Press JUMP at the peak to release and grab next rope
• Tree Climbing: Press JUMP to climb up, then fall to a new path
• Secret Tunnels: Press SLIDE when you see a dark hole on ground

LEVELS (5 total):
----------------
Level 1: Jungle - Easy, learn controls
Level 2: River - Rope swings appear
Level 3: Cave - Dark, bats, tree climbing
Level 4: Volcano - Fast, lava pits
Level 5: Treetops - Very fast, many ropes

HOW TO INSTALL:
---------------
1. Create a folder called "jungle-escape"
2. Create subfolder "levels"
3. Save each file with the exact name shown:
   - index.html
   - style.css
   - game.js
   - levels/level1.js
   - levels/level2.js
   - levels/level3.js
   - levels/level4.js
   - levels/level5.js
4. Open index.html in any web browser (Chrome, Firefox, Safari, Edge)
5. No internet needed after download!

TIPS FOR KIDS:
--------------
• Watch the monkey - when an obstacle approaches, press the correct button
• Green JUMP button = jump over things on ground
• Orange SLIDE button = duck under things above
• Look for dark holes on ground - that's a secret tunnel!
• On rope swings, wait until monkey is at the highest point, then press JUMP

HAVE FUN! 🐒🍌🌴
